# Mac OS에서 사전 사용하기

# 사용법

  • 포스터치의 기능 중 하나가 글자를 포스터치하면 사전검색이 가능하다.
  • 터치패드로는 "세 손가락으로 (가볍게) 클릭"이 있는데, 설정 > 트랙패드 메뉴에 가서 체크를 해줘야 한다.
  • 키보드로는 Ctl + CMD + D

# 사전 순서 조정 및 추가

  • Dictionary.app를 실행하면 검색할 사전의 종류와 순서 지정이 가능하다.
