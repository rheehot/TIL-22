# d3.js

  • Observable: 최근 (2018년 1월) 릴리즈된 d3용 노트북 서비스 👍
  • D3 blocks
  • D3 Discovery: d3 관련 페이지를 잘 모아놨다.
  • The Big List of D3.js Examples: 2000여개의 링크를 모아두었다. 깨진 링크가 많지만, 그래도 훑어보기에 유용하다.
  • d3.js 책 속 추천목록
    • DataWrapper: 최신브라우저에선 인터렉티브, 오래된 브라우저엔 정적 이미지로 표시한다. 저널리스트를 위해서 설계된 서비스.
    • dc.js: crossfilter를 이용한 개량된 버전인데, 물건이다.
    • billboard.js: naver 개발자가 만든 c3.js 개량버전. 2017년까지는 활발하게 개발했었다. D3.js v4를 지원한다.
  • Vega
  • Vega-lite: d3.js의 원산지였던 IDL의 라이브러리
    • Vega-lite 예제: 월별 강수량: 아래 코드를 이용하여 테스트할 수 있다.
{
  "$schema": "https://vega.github.io/schema/vega-lite/v3.json",
  "data": {"url": "https://api.taegon.kr/stations/108/?sy=2016&ey=2016"},
  "mark": "bar",
  "encoding": {
    "x": {"timeUnit": "month", "field": "Date", "type": "ordinal"},
    "y": {"aggregate": "sum", "field": "rainfall", "type": "quantitative"}
  }
}
  • xkcd styled chart lib

# 인포그래픽스

  • DataVisProject 👍
  • 인포그래픽스.kr
  • UW Interactive Data Lab
  • 중앙일보 스페셜 콘텐츠: 꽤나 오래 전부터 만들어온 걸 최근에야 발견했다. 중앙일간지여서 그런지 퀄리티가 상당히 좋다. 주제에 따라 주장이 편향되어있는 점이 아쉽지만, 우리나라에서 인포그래픽스를 이용한 양질의 스토리텔링을 주기적으로 뽑아낸다는 점이 북마크한 이유이다.
  • kepler.gl powered by Uber 👍 브라우저 상에 바로 자료를 올려서 맵을 만들 수 있고, 서버로 자료를 전송하지 않는다고 한다.
  • VIS.GL powered by Uber: 우버가 공개한 시각화 관련 라이브러리
  • UBER Moment: 우버가 자사 자료를 이용해서 지역별 도착 시간을 지도정보로 제공해준다.
  • OEC (The Observatory of Economic Complexity): 공개된 무역데이터를 이용해서 API를 제공하고 시각화를 하고 있다. 👍
  • from Data to Viz: 데이터 형태별로 그래프 선택법 같은 정보를 잘 정리해두었다. 처음 시작하기에 좋은 홈페이지.
  • Flourish: 2016년 전세계 해상물류 지도를 그렸던 그룹 👍
  • ResourceTrade.earth: 전세계 무역을 잘 시각화한 사이트
